# 📖 Personal Diary App

A simple full-stack Personal Diary web application built using HTML, CSS, JavaScript, Node.js, and Express. Users can register, log in, create diary entries, edit them, and delete them. Each user's diary entries are stored separately in a JSON database.

## ✨ Features

- User Registration
- User Login
- Add Diary Entries
- Edit Entries
- Delete Entries
- Separate entries for each user
- Responsive and modern UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- JSON (Database)

## 🚀 How to Run

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Start the server:

```bash
node server.js
```

4. Open your browser and visit:

```
http://localhost:3000
```
